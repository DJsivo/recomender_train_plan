# Рекомендательная система по физическим упражнениям

Курсовой проект: десктопное приложение на Python, которое по параметрам пользователя (пол, возраст, вес, уровень активности, цель тренировок, ограничения по здоровью и т.д.) формирует персональную программу тренировок с учётом медицинских противопоказаний и базовой ML‑модели по упражнениям.

## Постановка задачи

Цель проекта — разработать настольное приложение, которое:

- собирает профиль пользователя (антропометрия, уровень активности, цели тренировок, опыт, доступное оборудование);
- учитывает заболевания и специальные состояния (сердечно‑сосудистые, суставы, спина, ожирение, диабет, астма, беременность, после операций и др.);
- формирует персональный план тренировок на несколько недель, ограничивая нагрузку в зависимости от заболеваний и реального доступного времени;
- использует ML‑подход для ранжирования упражнений (content‑based рекомендация) на основе большого каталога реальных упражнений;
- сохраняет/загружает профиль и план в JSON и позволяет наглядно просматривать расписание тренировок.

## Стек

- Python 3.11
- PySide6 — десктопный UI
- NumPy, pandas — работа с табличными данными
- scikit-learn — ML‑компонент (TF‑IDF и ранжирование упражнений по целям пользователя)
- matplotlib — графики и визуализации

## Структура проекта

project_root/
  app/
    __init__.py
    main.py              # точка входа
    ui/                  # интерфейс
      __init__.py
      main_window.py
    core/                # логика, данные, ML
      __init__.py
      models.py          # dataclass-модели профиля, упражнений и плана тренировок
      data_access.py     # загрузка/сохранение JSON-файлов (профиль, план, упражнения)
      recommender.py     # генерация плана тренировок с учётом профиля и заболеваний
      exercise_importer.py  # импорт каталога упражнений из free-exercise-db
      conditions.py      # медицинский слой: заболевания и ограничения по нагрузке
      exercise_ml.py     # ML-компонент: TF-IDF и ранжирование упражнений
    data/                # JSON-файлы профиля, плана и упражнений
      README.md
      exercises.json     # объединённый каталог упражнений (свой + free-exercise-db)
      conditions.json    # перечень заболеваний и ограничений (медицинский слой)
  tests/                 # автотесты (по мере разработки)
    __init__.py
  requirements.txt
  README.md
  TESTING.md

   ```bash
   python -m venv .venv
   .venv\Scripts\activate
   ```

4. Установить зависимости:

   ```bash
   pip install -r requirements.txt
   ```

5. Запустить приложение:

   ```bash
   python -m app.main
   ```

   или

   ```bash
   python app\main.py
   ```

После запуска откроется главное окно приложения с несколькими вкладками:

- **Профиль** — ввод и редактирование параметров пользователя (пол, возраст, вес, рост, уровень активности, цель, опыт, формат тренировок, доступное оборудование, заболевания и специальные состояния, максимум тренировок в неделю) с валидацией и сохранением в JSON.
- **План тренировок** — генерация персонального плана тренировок на несколько недель на основе правил, медицинского слоя и ML‑ранжирования упражнений (TF‑IDF), отображение плана в виде таблицы по неделям и дням.
- **График** — визуализация количества тренировок по неделям на основе текущего плана.
Подробнее о сценариях проверки работы см. в файле `TESTING.md`.
